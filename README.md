## 📖 简介 | Introduction

**Pixoclip** 是一个集成了前沿 AI 技术的在线视频剪辑与创作平台。我们致力于通过 **Web 端多轨道非线性编辑** 体验，结合云端强大的算力，让创作者能够一站式完成从素材生成、编辑到渲染的全流程。

独特的 **云端数据库存储** 机制，确保您的每一次灵感都被安全缓存，随取随用，无需担心本地性能瓶颈。

## ✨ 核心特性 | Features

### 🖥️ 极致编辑体验
- **Web 端多轨道编辑**：媲美桌面级软件的剪辑体验，支持复杂的图层与音视频轨道管理。
- **双模预览**：
    - 🖼️ **原画模式**：所见即所得，精准把控细节。
    - ⚡ **极速模式**：低延迟流畅预览，提升剪辑效率。
- **云端原生**：所有的合成结果实时云端缓存，跨设备无缝协作。
- **🤖 Agent 助手 (Beta)**：全新的智能助手，辅助您完成复杂的创作任务。

### 🚀 强大的 AI 工具箱
我们内置了丰富的 AIGC 能力，赋能创作的每一个环节：

| 功能模块 | 详细说明 |
| :--- | :--- |
| **🎨 AI 生图/生视频** | 一键生成高质量素材，激发创作灵感。 |
| **🧩 智能元素分层** | 自动提取图像元素，完美支持 **透明通道 (Alpha Channel)**，并在 Web 端直接合成。 |
| **🗣️ 多语种 TTS** | 强大的文生语音引擎，支持多语种、多音色选择，情感丰富。 |
| **👤 数字人定制** | 对口型专家。支持**图片指定数字人**、**单/双音频对话**及**视频对口型**。 |
| **🎤 AI 变声** | 将输入语音转换为指定音色，同时完美保留原语速与语调。 |
| **📝 字幕识别** | 高精度自动语音转文字（ASR），快速生成字幕轨道。 |
| **🧹 智能消除** | 强大的 Inpainting 技术，支持字幕消除及自定义区域文字/物体消除。 |
| **✂️ 视频智能切分** | 懂内容的剪辑师。支持按**镜头**或**视频内容**自动拆分长视频。 |

## 🛠️ 技术规格 | Specs

平台全面支持主流的媒体格式，满足专业制作需求：

| 类型 | 支持格式 | 备注 |
| :--- | :--- | :--- |
| **视频** | `.mp4`, `.webm` | 完美支持 `.webm` 透明通道视频 |
| **音频** | `.wav`, `.mp3`, `.m4a` | 覆盖无损与主流压缩格式 |
| **字幕** | `.srt` | 标准外挂字幕格式 |
| **字体** | `.ttf` | 支持上传自定义字体文件 |
